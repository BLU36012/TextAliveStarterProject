# TextAliveStarterProject
This a starter project using the TextAlive App API to create a simple music video player that displays lyrics that syncs to the part of the song being played. It has a play and pause button.

To run the project:
-Clone the repoisotry
-Run `npm install` to install dependencies if you don't have pascal installed globally
-Run `npm start` to start the development server

Features:
- Displays the title and artist of the song
- Displays the lyrics of the song that syncs to the part of the song being played
- Video (if linked to a video) that syncs to the part of the song being played
- Play and pause button to control the music and video playback
- Uses the TextAlive App API to fetch song information and lyrics
- Gradient background that transitions smoothly between colors using CSS animations
